Dynamic DOE

A dynamic design of experiment (DOE)
that statistically analyzes a set of experiment data,
using neural networks (NN), in order to find how the
input features effects the outputs of the experimentation.
Specifically, finding which input features can optimize the
output in order to reduce the overall need of
extensive experimentation.
