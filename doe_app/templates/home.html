Dynamic DOE

A dynamic design of experiment (DOE) that statistically analyzes a set of experiment data, using neural networks (NN), in order to find how the input features effects the outputs of the experimentation. Specifically, finding which input features can optimize the output in order to reduce the overall need of extensive experimentation.

User Guide

The main purpose of the web application is to analyze dataset input with Artificial Intelligence techniques. These datasets would typically describe a collection of inputs for a given application, for ex. different print settings (input), for a 3D printer (application).

Datasets:

The user will have to load at least one dataset to access the graph visualization.

  • Navigate to the Datasets page via the top menu
  • Drag and drop/click to upload your file of choice.
  • Make sure the csv file matches the file format specified at the top of the page.
  • Click the 'refresh' button to enable a preview of the uploaded file (to check if it was formatted correctly)

Analyzing the datasets:

  • Navigate to the Graph page via the top menu
  • Click on the file you want to display
  • A graph displaying the effect of each input feature associated with the selected dataset will be displayed
  • On the graph, if you choose an input feature with a "C", you are choosing a continous feature.
    • A line chart will be displayed.
  • If you choose an input feature with a "D", you are choosing a discrete feature.
    • A bar chart will be displayed.
  • Use the left-hand side form to add an additional feature to the dataset (the page will be reloaded)
  • Use the right-hand side form to test the dataset with restrictions on specified input features
